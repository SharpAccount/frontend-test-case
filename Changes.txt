Список изменений по компонентам и стору:

Общее:
    1. Сдекомпозировал ui и store,
    2. Сделал re-export модулей для простоты воспроиятия импортов

Store:
    1. Разделил appSlice на slices по следующим сущностям:
        - User,
        - Product:
            1. Удалил state для error (у него не было применения)
        - Cart:
             1. Удалил избыточные поля в initialState
                (более точные изменение описаны в комментариях),
             2. Добавил утилитарную функцию обновления корзины
                во избежания дублирования кода,
             3. Изменил синтаксис очистки корзины (присвоением initialState)

UI:
    1. Везде заменил аргумент в useSelector на select... вместо стрелочной функции,
    2. Везде изменил вызов dispatch функции с dispatch({type: 'action'})
       на dispatch(action()) для лучшей типизации,
    3. Сделал кнопку изменения видимости фильтров рабочей
